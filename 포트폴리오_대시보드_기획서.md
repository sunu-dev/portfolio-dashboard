# 주식 포트폴리오 실시간 대시보드 기획서

**SOLB PORTFOLIO**
저점 매수 전략 기반 포트폴리오 관리 시스템

- 작성일: 2026년 3월 22일
- 배포 URL: https://solb-portfolio.netlify.app
- GitHub: https://github.com/sunu-dev/portfolio-dashboard

---

## 1. 프로젝트 개요

### 1.1 목적

미국 주식 포트폴리오의 저점 매수 전략을 실시간으로 모니터링하고, 이란 전쟁 등 지정학적 이벤트 대비 시장 변동을 추적하는 웹 기반 대시보드 시스템을 구축한다.

특정 1명에게 공유하기 위한 개인용 툴이며, 수동 개입 없이 자동으로 데이터가 갱신되는 것을 목표로 한다.

### 1.2 배경

- 2026년 2월 28일 이란 전쟁 발발로 미국 증시 급락
- 반도체/AI 섹터 전쟁 전 대비 10~15% 할인 → 저점 매수 기회
- 실시간 가격 모니터링 + 한국어 뉴스 + 기간별 분석 필요

### 1.3 대상 사용자

본인 + 공유 대상 1명 (총 2명). 모바일/PC 브라우저에서 URL 접속.

## 3. 핵심 기능

### 3.1 실시간 가격 모니터링

- Finnhub API를 통한 30초 간격 자동 갱신 (토/일 휴장일에는 금요일 종가 표시)
- 현재가, 전일 대비 등락률, 52주 고/저 범위 바 표시
- 목표가/손절가/매수구간 도달 시 알림 배너 자동 표시
- 매크로 지표: S&P 500, NASDAQ, KOSPI, USD/KRW 환율, WTI 유가, VIX

### 3.5 종목 관리 기능

- 티커 검색으로 종목 추가/삭제 (Finnhub search API)
- 단기/장기/관심 3개 카테고리 탭 전환
- localStorage로 포트폴리오 상태 유지 (브라우저 닫아도 복원)
- 설정 패널: API 키 변경, 자동 새로고침 토글, 갱신 주기 조절 (15/30/60초)

## 4. 기술 아키텍처

### 4.1 시스템 구성도

전체 파이프라인: 단일 HTML 파일 → GitHub → Netlify 자동 배포

| 구성요소 | 상세 |
|----------|------|
| 프론트엔드 | 단일 HTML 파일 (CSS + JS 인라인) |
| 호스팅 | Netlify (무료 플랜) |
| 소스 관리 | GitHub (sunu-dev/portfolio-dashboard) |
| 주가 API | Finnhub.io (무료 플랜, 60호출/분) |
| 환율 API | open.er-api.com (무료) |
| 뉴스 API | rss2json.com + Google News RSS (한국어) |
| 폰트 | Google Fonts — Noto Sans KR |
| 배포 | git push → Netlify 자동 빌드/배포 |

### 4.2 API 키 및 설정

| 항목 | 값 |
|------|---|
| Finnhub API Key | 하드코딩 (설정에서 변경 가능) |
| 자동 새로고침 | 30초 간격 (기본값) |
| 뉴스 갱신 | 10분 간격 |
| 디자인 테마 | 다크 모드 (배경 #0f1117) |
| 액센트 컬러 | #6C5CE7 (보라) |

## 6. 배포 및 운영

### 6.1 배포 파이프라인

GitHub 레포지토리에 push하면 Netlify가 자동으로 빌드하고 배포한다.

| 단계 | 상세 |
|------|------|
| 1. 코드 수정 | HTML 파일 수정 (로컬 또는 Cowork) |
| 2. Git Push | `git add . && git commit && git push` |
| 3. 자동 배포 | Netlify가 GitHub webhook 감지 → 빌드 |
| 4. 반영 확인 | solb-portfolio.netlify.app 접속 확인 |

## 7. 현재 한계 및 향후 계획

### 7.1 현재 한계점

| 한계 | 상세 | 영향도 |
|------|------|--------|
| 토/일 데이터 없음 | 미국 증시 휴장일에는 금요일 종가 표시 | 낮음 |
| 배포 수동 | HTML 수정 후 git push 필요 | 중간 |
| API 호출 제한 | Finnhub 무료 60회/분 → 종목 많아지면 지연 | 중간 |
| 뉴스 CORS | 일부 RSS 피드 CORS 차단 가능성 | 낮음 |
| 백엔드 없음 | 클라이언트 전용 → 데이터 저장/히스토리 불가 | 중간 |

### 7.2 향후 개선 계획

1. 백엔드 서비스 도입: 뉴스 자동 업데이트, 가격 알림 푸시 알림
2. 차트 추가: 종목별 가격 차트 (TradingView 위젯 또는 Chart.js)
3. 이벤트 트래커 확장: 이란 전쟁 외 추가 이벤트 등록 기능
4. GitHub Actions 자동화: 스케줄 기반 뉴스 업데이트 + 자동 커밋/배포
5. 모바일 최적화: PWA 전환으로 오프라인 접근 지원
